实验1 局域网的基本配置（一）

实验目的

  • 掌握局域网（LAN）的基本概念
  • 学会Windows 10下局域网的配置方法
  • 理解IP地址、子网掩码的作用

知识铺垫

  • 局域网是在小范围内（如办公室、教室）将计算机等设备互联的网络
  • IP地址是设备在网络中的“身份标识”，如 192.168.1.xxx 是常见的局域网IP段

实验1 局域网的基本配置（二）

操作步骤（简化）

  • 打开：控制面板 → 网络和Internet → 网络和共享中心
  • 点击更改适配器设置，找到当前连接的网络（如以太网）
  • 右键选择属性，双击Internet协议版本4（TCP/IPv4）
  • 配置IP地址（如 192.168.1.100）、子网掩码（默认 255.255.255.0），点击确定

验证方法

  • 打开命令提示符（cmd），输入：ipconfig 查看IP配置是否生效

实验2 文件与打印机共享设置与应用（一）

实验目的

  • 掌握Windows 10下文件和打印机共享的开启方法
  • 学会在局域网内访问共享资源

应用场景

  • 办公室内多人共享文件、打印机，提高办公效率

实验2 文件与打印机共享设置与应用（二）

文件共享步骤

  • 选择需要共享的文件夹，右键→属性 → 共享选项卡
  • 点击共享，选择要共享的用户（如Everyone），设置权限后点击共享
  • 其他计算机可通过网络 → 对方计算机名访问共享文件夹

打印机共享步骤

  • 打开设置 → 设备 → 打印机和扫描仪，选择要共享的打印机
  • 点击管理 → 打印机属性 → 共享选项卡，勾选共享这台打印机并设置共享名
  • 其他计算机可通过添加打印机 → 网络打印机找到并安装共享打印机

实验3 常用网络测试工具的应用（一）

常用工具

  • ping：测试网络连通性
  • ipconfig：查看本地网络配置
  • tracert：追踪数据包传输路径

工具作用

  • 帮助排查网络故障（如无法上网、连接不稳定等）

实验3 常用网络测试工具的应用（二）

操作示例

  • ping命令：打开 cmd，输入 ping www.baidu.com，查看是否有数据包返回（判断是否能连接互联网）
  • ipconfig命令：输入 ipconfig /all，可查看详细的IP、子网掩码、网关、DNS等信息
  • tracert命令：输入 tracert www.sina.com.cn，可查看数据包到新浪服务器经过的节点

故障排查思路

  • 先 ping 网关（如 192.168.1.1）判断局域网是否通
  • 再 ping 外网判断是否是宽带问题

单元2 总结

重点回顾

  • 局域网配置的核心是IP地址和子网掩码的设置
  • 文件/打印机共享是局域网资源协同的关键操作
  • 网络测试工具是排查网络问题的利器

学习建议

  • 多动手实操，在实验室或班级局域网环境中反复练习，巩固操作熟练
